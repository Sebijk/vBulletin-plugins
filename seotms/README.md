# Sebijks SEO für vBulletin 3.6-3.8

Letztes Update im Jahr 2008.

Dieses Produkt macht ihr vBulletin Forum mithilfe von TMS suchmaschinenfreundlicher.

1. Stellen Sie sicher, dass das Produkt Template Modification System von Andreas installiert ist. Wenn nicht, können Sie das unter https://www.vbulletin.org/forum/showthread.php?t=152931 herunterladen. Wenn Sie es nicht benutzen wollen, müssen Sie die Template-Änderungen manuell durchführen.

2. Importieren Sie das Produkt product-sebijk_seotms.xml

3. Laden Sie die htaccess.txt in ihrem Forum-Verzeichnis hoch und benennen Sie htaccess.txt in .htaccess um.

Nützliche Ergänzung: DCSEO Linkbacks: http://www.vbulletin.org/forum/showthread.php?t=157031

Wer die Rewrite-Urls von Sebijks SEO nicht mag, kann sie ausschalten, unter http://www.vbulletin.org/forum/showthread.php?t=173738 gibt es ein Add-on dazu, dass genau darauf spezialisiert ist und sich wunderbar im Kombination von Sebijks SEO einsetzen lässt.
